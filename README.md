# 🎯 Student Grade Calculator

## 📌 Project Overview
This project is a simple Python-based application that calculates a student's grade based on marks and displays an encouraging message.

## 🎯 Objectives
- Learn Python basics (input, functions, loops)
- Implement grading logic using if-elif-else
- Handle invalid inputs using loops
- Build a user-friendly console application

## 🛠️ Technologies Used
- Python

## ⚙️ Features
- Accepts student name and marks
- Validates marks (0–100 only)
- Calculates grade (A, B, C, D, F)
- Displays motivational message
- Handles invalid input using while loop

## 📝 Grading System
| Marks Range | Grade |
|------------|-------|
| 90–100     | A     |
| 80–89      | B     |
| 70–79      | C     |
| 60–69      | D     |
| 0–59       | F     |
